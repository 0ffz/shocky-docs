# Admonitions

## Single line admonitions

```markdown
Admonition text `with formatting` {.info}
```

Admonition text `with formatting` {.info}

## Multiline admonitions

```markdown

| **Admonition title**
|
| More text
| - a
| - b
| ```kotlin
| fun main() {
|     println("Hello, world!")
| }
| ```

{.info}
```
